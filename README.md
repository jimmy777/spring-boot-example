# Spring Boot Example

Example application using spring boot.

 - Startup with automatic configuration
 - Logging example
 - Local H2 file based database
 - Using flyway migrations
 - Using spring jpa repository
 - Jpa tests
 - Mock mvc tests
 - Integration tests with real server
